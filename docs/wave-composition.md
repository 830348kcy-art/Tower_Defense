# 웨이브 구성표

이 문서는 `Data/StageCatalog.cs`의 현재 생성 규칙을 기준으로 정리한 PR 검토용 웨이브 구성표입니다.
표의 숫자는 각 스테이지에서 직접 스폰되는 적의 총량입니다. `분열체`가 사망하며 만드는 `작은 분열체`는 표 수량에 별도로 더하지 않습니다.

## 적 구성

| 구분 | 적 |
|---|---|
| 일반 | 일반, 빠른, 분열체, 작은 분열체 |
| 엘리트 | 엘리트, 돌진 엘리트, 재생 엘리트, 유령 엘리트 |
| 중간보스 | 중간보스, 돌진 중간보스, 분열 중간보스, 속도 중간보스 |
| 보스 | 보스, 돌진 보스, 분열 보스, 속도 보스 |

총 16종입니다. 엘리트(저항)과 챕터별 슬라임/야수/수정/어둠 변종은 사용하지 않습니다.

## 실제 8웨이브 배분 규칙

스테이지별 구성표의 총량을 아래 웨이브에 나누어 배치합니다.

| 적 묶음 | 배치 웨이브 |
|---|---|
| 일반 | W1, W2, W3, W8 |
| 빠른 | W2, W3, W6, W8 |
| 분열체 | W3, W4, W8 |
| 엘리트 | W4, W7, W8 |
| 돌진 엘리트 | W5, W7, W8 |
| E.기타 | W6, W8 |
| 중간보스 / 보스 | W8 |
